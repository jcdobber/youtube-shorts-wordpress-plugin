# YouTube Shorts Gallery

Deze WordPress plugin maakt het mogelijk om in de beheeromgeving een lijst met YouTube Shorts vast te leggen en deze met een shortcode aan de voorkant te tonen. De bezoekers krijgen een raster met kaarten die dezelfde stijl benaderen als het aangeleverde voorbeeld. Bij het aanklikken opent een modale popup met daarin de ingesloten YouTube-video.

## Installatie
1. Kopieer de map `youtube-shorts-gallery` naar de `wp-content/plugins` map van je WordPress-installatie (of pak de inhoud uit in een map met dezelfde naam).
2. Activeer de plugin in het WordPress beheerderspaneel via **Plugins** → **Geïnstalleerde plugins**.

## Gebruik
1. Ga in de backend naar **YouTube Shorts**.
2. Voeg voor iedere Short de URL, titel, korte omschrijving en (optioneel) de duur toe en sla op.
3. Plaats de shortcode `[youtube_shorts_gallery]` in een pagina of bericht om de galerij te tonen.

De plugin genereert automatisch de thumbnails op basis van de video-URL en toont een popup met de video wanneer een kaart wordt aangeklikt.

> Tip: bekijk je de repository op GitHub? Navigeer naar de map `youtube-shorts-gallery` om alle pluginbestanden te zien.
